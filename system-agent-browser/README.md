# Agent Browser

Agent Browser contributes the `agent_browser` tool for controlling browsers, web apps, iOS Safari, and CDP-connected Electron apps through the external `agent-browser` CLI.

This is an optional installable extension, not a bundled system extension. Users install it from **Settings → Extensions → Install**; after installation it loads from runtime state as a normal user extension and is disabled until enabled.

Use it for autonomous development validation, app smoke checks, repeatable browser automation, and product-under-test screenshots. For the shared in-app Workbench Browser, use the Browser extension instead.
